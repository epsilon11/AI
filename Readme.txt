How to execute code in files
----------------------------

1. Install node.js or brew install node
2. mkdir NeuralNetworkIn5mins
       cd NeuralNetworkIn5mins
       npm init
3. npm install brain.js
4. Add any of the files to NeuralNetworkIn5mins directory
5. Go to NeuralNetworkIn5mins and type "node <any of the file name in the directory>"". E.g - "node lebronCavs.js"
